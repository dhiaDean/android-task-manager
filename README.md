## Project Overview

This is an Android application designed to help users manage their tasks efficiently. The application provides functionalities for user registration, login, and comprehensive task management, allowing users to create, view, and update tasks. It supports categorizing tasks as 'Personal' or 'Public' and tracking their status as 'Completed' or 'Pending'.

## Features

*   **User Authentication:** Secure registration and login system.
*   **Task Creation:** Users can add new tasks with details.
*   **Task Viewing:** Display a list of all tasks.
*   **Task Details:** View detailed information for each task.
*   **Task Status Management:** Mark tasks as 'Completed' or 'Pending'.
*   **Task Categories:** Organize tasks into 'Personal' and 'Public' categories.
*   **Navigation Drawer:** Easy navigation throughout the application.

## Technologies Used

*   **Android SDK:** For building the native Android application.
*   **Java:** The primary programming language.
*   **SQLite Database:** For local data storage and management (`DBHelper.java`, `DatabaseHelper.java`).

## Installation

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Android Studio (latest version recommended)
*   Android SDK
*   Java Development Kit (JDK)

### Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dhiaDean/android-task-manager.git
    ```
2.  **Open in Android Studio:**
    *   Launch Android Studio.
    *   Click on `Open an existing Android Studio project`.
    *   Navigate to the cloned repository directory and select it.
3.  **Sync Gradle:**
    *   Let Android Studio sync the Gradle project. If prompted, update any outdated plugins or SDK components.
4.  **Run on Emulator or Device:**
    *   Connect an Android device or start an AVD (Android Virtual Device) emulator.
    *   Click the 'Run' button (green triangle) in Android Studio to install and launch the app.

## Usage

1.  **Register:** Create a new user account.
2.  **Login:** Use your credentials to access the app.
3.  **Home Screen:** View your tasks. You can navigate through categories using the drawer menu.
4.  **Add Task:** Use the appropriate button to add a new task, providing a title, description, category, and initial status.
5.  **View/Edit Task:** Click on a task to see its details and make updates.
